# Portfolio - Sithija Oshan

A modern, responsive personal portfolio website built with React, showcasing my projects, skills, and professional journey as a Computer Science undergraduate and aspiring software developer.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Interactive UI**: Dynamic rotating text animations, smooth scrolling navigation, and expandable project cards
- **Project Showcase**: Detailed project information with technology stacks and GitHub links
- **Skills Visualization**: Interactive skill bars showing proficiency levels across different technologies
- **Professional Sections**: About, Education, Projects, Skills, and Contact sections
- **Mobile-First**: Optimized mobile navigation with hamburger menu

### Installation

1. Clone the repository:
```bash
git clone https://github.com/OshanDAS/Portfolio.git
cd Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
